# IP-Address
This Tool outputs your public &amp; private ip address.

<h3>I recommend you to Run all commands as sudo</h3>
<code>
sudo apt-get install python3\n
</code>
<br>
<code>
sudo apt-get install pip
</code>
<br>

<code>
sudo pip install -r requrements.txt
</code>
<br>

<code>
sudo python3 ip.py
</code>
